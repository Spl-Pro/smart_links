Odoo All in One Approval Level Workflow App

Odoo All in One Approval Level Workflow App helps to Manage Dynamic Approval Workflow On Sale Order Request Approval, Purchase Order Request Approval. This App Provides A Complete Approval Process. i.e Submit Request, Approve Request, Reject Request, Cancel Request, Rework On Rejected Request etc. You Can Also have the Option to View the Dashboard of the Approval Request and the User Can Easily Create A Request from the Dashboard, Also Edit the Approval Type from the Dashboard and Show the Number of Request Reviews in the Dashboard.
